devlog: install

cargo

cargo install devlog

vim syntax highlighting

To enable syntax highlighting of devlog entries in vim, you will need this directory on vim's runtime path. You can download the files to your $HOME/.vim directory: 

mkdir -p $HOME/.vim/ftdetect && \
mkdir -p $HOME/.vim/syntax && \
curl https://raw.githubusercontent.com/wedaly/devlog/master/integrations/vim/ftdetect/devlog.vim > $HOME/.vim/ftdetect/devlog.vim && \
curl https://raw.githubusercontent.com/wedaly/devlog/master/integrations/vim/syntax/devlog.vim > $HOME/.vim/syntax/devlog.vim
