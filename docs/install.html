devlog: install

command-line tool

  • .deb Package (Debian/Ubuntu): wget https://github.com/wedaly/devlog/releases/download/1.0.2/devlog_1.0.2_amd64.deb && sudo dpkg -i devlog_1.0.2_amd64.deb
  • .rpm Package (Fedora): wget https://github.com/wedaly/devlog/releases/download/1.0.2/devlog-1.0.2-1.x86_64.rpm && sudo dnf install devlog-1.0.2-1.x86_64.rpm
  • Homebrew (macOS): brew tap wedaly/devlog && brew install devlog
  • Cargo (Rust): cargo install devlog
  • Binaries: available for Linux and macOS

vim syntax plugin

The devlog-vim plugin provides syntax highlighting for devlog files. It can be installed as a Vim8 package:

git clone https://github.com/wedaly/devlog-vim ~/.vim/pack/plugins/start/devlog-vim
